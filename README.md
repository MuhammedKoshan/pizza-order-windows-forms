# 🍕 Pizza Order App (C# Windows Forms)

A simple desktop application built using **C# and Windows Forms** that allows users to create and customize their pizza order.

## 🚀 Features
- Select pizza size (Small, Medium, Large)
- Choose multiple toppings
- Select crust type (Thin / Thick)
- Choose dining option (Eat In / Take Out)
- Dynamic total price calculation
- Order confirmation system
- Reset form functionality

## 🛠️ Technologies Used
- C#
- .NET Framework
- Windows Forms (WinForms)

## 💡 How It Works
The application updates the order summary and total price in real-time based on user selections using event-driven programming.

## 📂 Project Structure
- UI built with WinForms Designer
- Logic handled in `Form1.cs`
- Uses control properties like `Tag` to store pricing

## ▶️ How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the application

## 📌 Notes
- Prices are stored inside control `Tag` properties
- Total price is calculated dynamically
